# Proyecto de Código Morse con Arduino

Proyecto desarrollado con Arduino para convertir texto a código Morse y reproducirlo mediante un buzzer, además de permitir el ingreso manual de señales Morse usando un botón físico.

## Descripción

Este sistema ofrece dos modos de funcionamiento:

1. **Modo de reproducción**  
   El usuario escribe una palabra desde el monitor serial y el sistema la convierte a código Morse para reproducirla con un buzzer.

2. **Modo de entrada manual**  
   El usuario utiliza un botón para enviar pulsaciones cortas y largas, que el sistema interpreta como puntos y guiones del código Morse.

## Componentes utilizados

- Arduino
- Buzzer
- Botón
- Monitor serial

## Funcionalidades principales

- Conversión de texto a Morse
- Conversión de secuencias Morse a caracteres
- Reproducción sonora con buzzer
- Lectura de pulsaciones desde botón
- Interacción por monitor serial
